# ChattyKat

ChattyKat is a lightweight Java-based chat application demonstrating classic socket programming using TCP.  
It includes both a server application and a Swing-based client application, each of which can be packaged as standalone JAR files. Multiple clients can connect simultaneously, send messages, and receive broadcasts from other connected users.
